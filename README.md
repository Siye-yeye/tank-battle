# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 制作的经典坦克大战网页游戏。

## 🎯 玩法

- 控制你的绿色坦克，消灭所有敌方红色坦克
- 利用砖墙作为掩护（可被子弹摧毁）
- 钢墙无法被摧毁，善加利用
- 每波敌人会越来越多，越来越强

## 🎮 操作方式

| 按键 | 操作 |
|------|------|
| `W A S D` 或 `↑ ↓ ← →` | 移动坦克 |
| `空格键` | 发射子弹 |
| `P` | 暂停 / 继续 |
| `ENTER` | 开始游戏 / 重新开始 |

## 🚀 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/YOUR_USERNAME/tank-battle.git
   ```
2. 直接用浏览器打开 `index.html` 即可开始游戏

或者直接访问 GitHub Pages 在线游玩！

## ✨ 特性

- 🎨 像素风格画面，粒子爆炸特效
- 🧱 可破坏的砖墙 + 不可破坏的钢墙
- 🤖 敌人 AI 会追踪玩家并射击
- 📈 波次递增难度
- 💥 屏幕震动效果
- 🗺️ 小地图显示敌我位置
- ⏸️ 暂停功能

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架依赖）
- CSS3 动画
